Experiment 04: Infinite Worlds

Description

This is a demo of a 2d world generation algorithm to make infinite worlds from noise.

Technical

In this demo, I use perlin noise averaged to create interesting terrain. I use some other samples for determining the biome and wind. I generate curvature from the deltas of the heights.

Reflection

This was a really fun project. It was fun to make such great progress, since I have always wanted to make similar projects.
