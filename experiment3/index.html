Experiment 3 : Alternate Worlds


Description

This is an experiment to make an implementation of autotile and create a map editor that converts text to a map.

Technical

This uses the autotile algorithm which uses bitmasking to determine how each of the cliff or tree tiles should be placed.

Reflection

The autotiles came out alright. This tileset is not perfect, and I could not figure out where each of the tiles were supposed to go, but it looks mostly right. Due to the seeded nature of the map, the snow does not work properly.
